# Qatar Ground Motion Service
The Qatar Ground Motion Service (QGMS) is a national platform that measures and maps land deformation across Qatar using precise InSAR data. It tracks how the ground surface moves over time to support infrastructure management, environmental monitoring, and geohazard assessment.
Current deployment: [https://www.esrs.wmich.edu/webmap/qatar/](https://www.esrs.wmich.edu/webmap/qatar/)
